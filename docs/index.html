Indoor Positioning plugin for Cordova

Kabisa developed a Cordova plugin that wraps Signify's SDK for indoor positioning. This plugin is publicly available for anyone that wants to develop cross-platform mobile apps using Cordova.

This plugin requires Signify's indoor positioning SDK for Android and iOS, which we don't distribute. Contact Signify for more information concerning these SDKs.
